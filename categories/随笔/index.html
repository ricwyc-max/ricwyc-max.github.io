🔍 如何搜索文章？

📱 在电脑上搜索

Windows 系统

  1. 按下键盘上的 Ctrl + F 键
  2. 在右上角出现的搜索框中输入关键词
  3. 浏览器会自动高亮显示匹配的内容
  4. 使用 F3 键查找下一个匹配项

Mac 系统

  1. 按下键盘上的 Cmd + F 键
  2. 在右上角出现的搜索框中输入关键词
  3. 浏览器会自动高亮显示匹配的内容
  4. 使用 Cmd + G 键查找下一个匹配项

📲 在手机上搜索

iPhone / iPad

  1. 点击浏览器右上角的 分享图标 (📤)
  2. 向下滑动找到并点击 在页面上查找
  3. 在底部的搜索框中输入关键词

Android 手机

  1. 点击浏览器右上角的 三个点 (⋮)
  2. 选择 在页面中查找
  3. 在出现的搜索框中输入关键词

🗂️ 推荐的搜索步骤

1. 先浏览文章列表

访问 所有文章页面，这里有完整的文章目录。

2. 按标签筛选

查看 标签页面，可以根据主题快速找到相关文章。

3. 使用 Ctrl+F 搜索

在任何页面按下 Ctrl+F，输入：

  • 文章标题中的关键词
  • 作者名
  • 技术名词（如：Python、深度学习）
  • 项目名称

💡 搜索技巧

精确搜索

  • 使用引号搜索完整短语："计算机视觉"
  • 使用减号排除词语：Python -基础

快速定位

  1. 打开 所有文章页面
  2. 按下 Ctrl+F
  3. 输入想找的关键词
  4. 按 Enter 跳转到第一个匹配项
  5. 使用 F3 (Windows) 或 Cmd+G (Mac) 查找下一个

🌐 浏览器兼容性

浏览器 快捷键 位置
Chrome Ctrl+F 右上角
Firefox Ctrl+F 左下角
Edge Ctrl+F 右上角
Safari Cmd+F 右上角
